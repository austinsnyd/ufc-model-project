Using Machine Learning to Predict UFC Match Outcomes

üîç Overview

This project applies machine learning models to predict UFC fight outcomes using historical data and statistical features. Drawing from over 5,000 fights, the analysis explores how attributes such as striking accuracy, takedown percentage, betting odds, and fighter experience contribute to win probabilities. Models include Logistic Regression, Random Forest, and XGBoost ‚Äî each evaluated based on predictive performance, interpretability, and AUC scores.

Betting odds were the strongest predictor of outcomes, followed by striking accuracy and experience. While all models performed moderately well, Random Forest achieved the highest test accuracy (67.4%) and strongest AUC (.7228). Challenges included red-corner win bias, class imbalance, and limited granularity of fight data. Future work could include deep learning models and odds-independent analysis for sports betting use cases.
